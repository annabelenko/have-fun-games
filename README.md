# Have Fun Games

## Requirements

- Node.js

## Setup

### Frontend

1. Set current working directory to the frontend.
2. Install dependencies with `npm install`.

To start, run `npm run dev` in the frontend directory or use the VS Code launch configuration.

### Backend

1. Set current working directory to the backend.
2. Install dependencies with `npm install`.

To start, run `npm run dev` in the backend directory or use the VS Code launch configuration.
